OJ Visual SQL Builder

An ExtJs web app that makes constructing SQL as easy as drag 'n drop

First, Thanks to Martin Orth

Mr. Orth created the original version of this app, which you can see demo of here. His version is generally excellent, but incomplete. The purpose of this project is to carry the original to fruition.

Getting Started

The app doesn't yet have a build process or any scaffolding, but if you simply pull the repo and add it to your local web server--you should be able to run it without any snags. Since the app uses ExtJs, which in turn depends upon an async resource loader, you'll run into problems trying to run it directly from the file system. You can also run the latest version at the C9 instance.
